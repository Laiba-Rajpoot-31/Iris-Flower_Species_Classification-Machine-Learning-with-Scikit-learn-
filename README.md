# 🌸 Iris Flower Species Classification A complete ML pipeline to classify Iris flowers (Setosa, Versicolor, Virginica) using Python and Scikit-learn. ## 📌 What's inside - Exploratory Data Analysis (pairplot, descriptive stats) - Preprocessing: Label Encoding + StandardScaler - Stratified 80/20 train-test split - 4 classifiers: Random Forest · SVM (RBF) · KNN · Logistic Regression - 5-fold Cross-Validation + Confusion Matrices - Feature Importance (Random Forest) - Prediction on new samples ## 📊 Results | Model | Test Accuracy | |---|---| | SVM (RBF) | 96.7% ✅ | | KNN (k=5) | 93.3% | | Logistic Regression | 93.3% | | Random Forest | 90.0% | ## 🛠️ Tech Stack `Python` `Scikit-learn` `Pandas` `NumPy` `Matplotlib` `Seaborn` ## 💡 Key Findings - Setosa is perfectly linearly separable — all models classify it with 100% accuracy - Petal measurements account for ~87% of feature importance - SVM with RBF kernel is the best overall model
